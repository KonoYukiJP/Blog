Linux コマンド一覧

  • pwd - 作業ディレクトリ名を表示
  • ls - ディレクトリの内容を一覧表示する
  • cat ファイル - ファイルを表示する
  • mkdir ディレクトリ名 - ディレクトリを作成する
  • cd ディレクトリ - 現在のディレクトリを変更する
  • cp ソースファイル ターゲットファイル - ファイルをコピーする
  • mv ソース ターゲット - ファイルを移動する
  • open ファイル名 - ファイルとディレクトリを開く
  • rm ファイル - ディレクトリエントリを削除する
  • vi ファイル - プログラマーのテキストエディタ
  • nano ファイル - シンプルなテキストエディタ
